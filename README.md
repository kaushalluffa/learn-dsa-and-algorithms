# 📚 DSA Mastery Hub — NeetCode 150 Roadmap

A single-page interactive learning dashboard for mastering Data Structures and Algorithms, built around the [NeetCode 150](https://neetcode.io/roadmap) roadmap. Designed for visual learners who need to **see a data structure in action** before solving problems with it.

## ✨ Features

- **18 categorized sections** matching the NeetCode roadmap order
- **150 problems** sorted easy → hard within each category, with direct links to LeetCode and NeetCode video solutions
- **Learning resources** for each data structure (JavaScript-focused articles, videos, and implementations)
- **Real-world examples** showing where each data structure is actually used (DOM trees, TCP sliding window, git diff, etc.)
- **Interactive progress tracking** with checkboxes and a live progress bar
- **Pattern tags** identifying the algorithmic technique for each problem
- **Time estimates** for each problem

## 🚀 Usage

Just open this [URL](https://kaushalluffa.github.io/learn-dsa-and-algorithms/) or clone the repo to run on ur local in any browser. No build step, no dependencies, no server required. On your local just open the index.html with live server extension in vscode or open with browser.

## 📸 Preview

Open `index.html` locally to see the full interactive dashboard.
